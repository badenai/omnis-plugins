*   **Data Pipeline Architecture**: Time-series databases such as [QuestDB](concepts/infrastructure-data-pipeline.md) are critical. They utilize ASOF JOINs for temporal alignment of asynchronous tick data and SIMD-accelerated SQL for computing multi-dimensional Limit Order Book (LOB) arrays with sub-millisecond latency.

[finmlkit](concepts/numba-hft-microstructure-bars.md) using Numba JIT compilation provides >100x speedups for mapping [Order Flow Imbalance (OFI)](concepts/tick-level-order-imbalance-depth.md).
